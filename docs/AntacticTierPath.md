# Antarctic Tier 1 Pathway
**Immediate Ground-Based Validation for POTS**

**Sherry Moore**  
March 29, 2026

**Objective**  
Deploy POTS in a real extreme-environment analog to validate the six predictions and generate off-policy evaluation data before lunar deployment.

**Platform**  
U.S. Antarctic Program (USAP) / National Science Foundation rotating-crew stations (McMurdo, South Pole, Palmer, etc.).

**Protocol**  
- Small cohort of consenting volunteers (scientists and support staff)  
- 6–12 month continuous logging of authentic behavioral signal  
- Use existing multi-vendor equipment and comms constraints  

**Success Metrics**  
- ≥30 % reduction in crew onboarding time between rotations  
- ≥25 % improvement in knowledge retention  
- ≥40 % faster detection of operational/emotional drift  

**Timeline**  
Can begin Q3 2026 using existing NSF infrastructure or no-cost MOUs.

This Tier 1 analog provides the immediate, domestic proof-of-concept NASA reviewers require before CLPS integration.
