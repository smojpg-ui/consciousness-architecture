# POTS – Persistent Operational Twin System
**Executive Summary for NASA Ignition RFI**

**Sherry Moore**  
March 29, 2026

POTS is a zero-mass, software-only payload that provides longitudinal relational intelligence for lunar surface operations.

**Problem**  
Multi-vendor CLPS architectures and rotating crews cause operational knowledge to evaporate with every handoff.

**Solution**  
POTS creates a living **Relational Twin** of each crew member and the robotic ecosystem using contextual bandit dynamics on authentic behavioral signal.

**Key Features**
- Zero launch mass — runs on existing CLPS hardware  
- Edge sovereignty — no raw data leaves the base  
- Cross-rotation continuity — robotic assets inherit workflow and system knowledge  

**Validation Path**  
- Tier 1: U.S. Antarctic Program analog (immediate)  
- Tier 2: CLPS lander deployment (2027)  

POTS directly supports NASA’s accelerated CLPS cadence and permanent lunar outpost goals. The proposing team is self-funded and will deliver the completed software package and perform all operations at no cost to NASA.

Ready for inclusion in the Lunar Payload Database.
